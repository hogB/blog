## 异步



### 回调地狱

> 回调地狱是什么？它真的是一个地狱吗

这里给了一个需求需要我们在1s后输出cxy，2s后输出zrx，3s后输出wb

```js
setTimeout(() => {
    console.log(1)
    setTimeout(() => {
        console.log(2)
        setTimeout(() => {
            console.log(3)
        },3000)
    },2000)
},1000)
```

以上就是典型的回调地狱，不停地回调，一层套一层

而用生成器解决呢

```js
function one() {
    setTimeout(() => {
        console.log("111")
        iterator.next()
    },1000)
}
function fn() {
    yield one()
}
```
